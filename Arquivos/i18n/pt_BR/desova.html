Mais aplicativos aguardam.

Ligue o seu computador à Internet para explorar uma grande variedade de aplicações para Horizon OS.

Uma vez online, você será capaz de baixar e instalar software experimentado e testado aqui. Nossas escolhas garantem que os recursos do software se integrem bem na área de trabalho do Horizon OS.

Desculpe, a Loja Horizon OS não conseguiu estabelecer uma conexão com o servidor de repositório do Ubuntu e outros externos. Por favor, verifique sua conexão e tente novamente.

Conecte-se.

Eis algumas coisas que você pode verificar, dependendo do tipo de conexão que você tem:

Conexão com fio
  • O cabo está corretamente plugado?
  • O roteador está ativo e outros dispositivos podem acessar a rede? Tente reiniciar o roteador.
Conexão Sem-Fio
  • Você digitou corretamente a senha da rede sem-fio a partir do indicador de rede no canto superior direito?
    • Quando desconectado, o applet se parece como:
  • Existe um botão físico que pode ter acidentalmente desligado funcionalidade sem fio?
  • Existe um interruptor macio (teclado) que possa desligar?
    • Talvez você precise segurar a tecla FN enquanto pressiona a tecla de função.
    • Aqui está um exemplo
    • Nada funciona ou a conexão está lenta?

    Lamento ouvir isso. Você vai precisar de uma conexão com fio temporária para instalar os drivers corretos.

    • O guia pode ter eles disponíveis para o seu sistema
    • Caso contrário, você precisará instalar manualmente drivers de terceiros para o seu hardware, ou um pacote específico contendo o firmware. Veja no tópico Drivers em para obter mais detalhes.
    • Sinta-se livre para se você precisar de ajuda.

    Correções diversas

    Essa seção contém operações que podem corrigir problemas comuns que você poderá deparar-se ao atualizar ou instalar um novo Software.

    Listas de Pacotes Desatualizada?

    Sua lista de repositórios talvez esteja desatualizada, o que pode causar Inúmeros Erros e Versões Desatualizadas de informação enquanto tentar instalar versões mais novas de softwares. Esse caso em particular acontece quando o Horizon OS conecta-se à internet pela primeira vez após a instalação.

    Atualiza as listas de repositórios:
    sudo apt update

    Pacotes Quebrados

    Quando uma instalação ou remoção anterior foi interrompida (por exemplo, devido à falha de energia ou perda de conexão com a Internet), mais programas não podem ser adicionados ou removidos sem a devida reconfiguração desses pacotes quebrados. Se necessário, você também precisará resolver as dependências e instalar quaisquer pacotes que faltam para que o software seja executado corretamente.

    Configure os pacotes que foram descompactados mas ainda não foram configurados:
    sudo dpkg --configure -a
    Baixar e instalar as dependências quebradas:
    sudo apt --fix-broken install

    Além disso, estão listados abaixo comandos de terminal equivalentes que, caso contrário, aparecem no Atualizador de Software

    Atualize todos os pacotes que tem uma nova versão disponível:
    sudo apt full-upgrade
    Atualize para uma nova versão do Horizon OS:
    sudo do-release-upgrade
